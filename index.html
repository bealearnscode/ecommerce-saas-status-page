Intro

This imaginary SaaS project was built using HTML5, CSS3, vanilla JavaScript, and jQuery. I chose jQuery for two reasons:

  1. I know how to extract JSON data with jQuery
  2. I believe a framework like React would have taken too long and I wanted to produce an MVP in the time allotted.
